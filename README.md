# AfyaLink Patients Portal

## Introduction
The AfyaLink Patients Portal is a web-based application built with Laravel 13 that enables patients to securely interact with the AfyaLink Hospital ERP System. It provides a self-service interface for accessing medical services, personal health information, and communication tools.

---

## Core Features

- Patient Registration and Profile Management  
- Appointment Booking and Scheduling  
- Access to Lab Test Results  
- Account Charges and Billing Overview  
- Prescription Tracking  
- Communication with Doctors  
- Access to Medical Records  

---

## Additional Capabilities

- Secure authentication and session management  
- Real-time synchronization with the ERP system  
- Appointment history and status tracking  
- Downloadable reports (lab results, prescriptions, invoices)  
- Notifications and alerts for appointments and updates  
- Patient dashboard with summarized health and activity data  
- Search and filtering for records and history  
- Responsive design for desktop and mobile access  

---

## Integration

The portal is tightly integrated with the AfyaLink ERP system (Microsoft Dynamics 365 Business Central) using NTLM authentication. This ensures:

- Secure data exchange  
- Consistent and up-to-date information  
- Centralized data management across systems
